# GB/T 7714-2015 参考文献著录规则

## 概述

GB/T 7714-2015《信息与文献 参考文献著录规则》是中国国家标准，规定了参考文献的著录项目、著录顺序、著录符号等。

## 文献类型标识

| 类型 | 标识代码 | 示例 |
|------|----------|------|
| 普通图书 | M | 专著、教材 |
| 会议录 | C | 会议论文集 |
| 汇编 | G | 文集、选集 |
| 报纸 | N | 报纸文章 |
| 期刊 | J | 期刊论文 |
| 学位论文 | D | 博士/硕士论文 |
| 报告 | R | 科技报告 |
| 标准 | S | 技术标准 |
| 专利 | P | 专利文献 |
| 数据库 | DB | 数据库 |
| 计算机程序 | CP | 软件 |
| 电子公告 | EB | 电子公告 |
| 电子资源 | EB/OL | 在线电子资源 |
| 档案 | A | 档案 |
| 地图 | CM | 地图 |
| 数据集 | DS | 数据集 |
| 其他 | Z | 其他未说明类型 |

## 著录格式详解

### 1. 专著 [M]

**格式：**
```
[序号] 主要责任者。题名：其他题名信息 [文献类型标识/文献载体标识]. 其他责任者。版本项。出版地：出版者，出版年：引用页码.
```

**示例：**
```
[1] 周志华。机器学习 [M]. 北京：清华大学出版社，2016.
[2] GOODFELLOW I, BENGIO Y, COURVILLE A. Deep Learning[M]. Cambridge: MIT Press, 2016.
[3] 陈登鳌。建筑抗震设计 [M]. 2 版。北京：中国建筑工业出版社，2015: 50-55.
```

### 2. 期刊论文 [J]

**格式：**
```
[序号] 主要责任者。题名 [J]. 期刊名，年，卷 (期): 起止页码.
```

**示例：**
```
[1] 张三，李四，王五。深度学习研究进展 [J]. 计算机学报，2024, 47(3): 123-135.
[2] VASWANI A, SHAZEER N, PARMAR N, et al. Attention Is All You Need[J]. Advances in Neural Information Processing Systems, 2017, 30: 5998-6008.
```

### 3. 会议论文 [C]

**格式：**
```
[序号] 主要责任者。题名 [C]//会议录名。出版地：出版者，出版年：起止页码.
```

**示例：**
```
[1] 李明。基于深度学习的图像识别 [C]//中国计算机大会论文集。北京：电子工业出版社，2023: 45-52.
[2] HE K, ZHANG X, REN S, et al. Deep Residual Learning for Image Recognition[C]//Proceedings of the IEEE Conference on Computer Vision and Pattern Recognition. 2016: 770-778.
```

### 4. 学位论文 [D]

**格式：**
```
[序号] 主要责任者。题名 [D]. 保存地点：保存单位，年份.
```

**示例：**
```
[1] 王小明。基于神经网络的自然语言处理研究 [D]. 北京：清华大学，2023.
[2] SMITH J. Deep Learning for Healthcare[D]. Stanford: Stanford University, 2022.
```

### 5. 专利 [P]

**格式：**
```
[序号] 专利申请者或所有者。专利题名：专利国别，专利号 [P]. 公告日期或公开日期.
```

**示例：**
```
[1] 华为技术有限公司。一种神经网络加速方法：中国，CN112345678[P]. 2023-01-15.
```

### 6. 标准 [S]

**格式：**
```
[序号] 主要责任者。标准编号，标准名称 [S]. 出版地：出版者，出版年.
```

**示例：**
```
[1] 国家标准化管理委员会。GB/T 7714-2015, 信息与文献 参考文献著录规则 [S]. 北京：中国标准出版社，2015.
```

### 7. 电子资源 [EB/OL]

**格式：**
```
[序号] 主要责任者。题名 [EB/OL]. (更新日期)[引用日期]. 获取和访问路径.
```

**示例：**
```
[1] 教育部。高等学校毕业设计（论文）指导手册 [EB/OL]. (2020-05-15)[2024-02-28]. http://www.moe.gov.cn/xxx.
[2] WIKIPEDIA. Artificial Intelligence[EB/OL]. [2024-02-28]. https://en.wikipedia.org/wiki/Artificial_intelligence.
```

## 作者著录规则

### 中文作者
- 3 人及以下：全部列出，用逗号分隔
- 超过 3 人：前 3 人 + ", 等"

**示例：**
```
张三，李四，王五。
张三，李四，王五，等。
```

### 英文作者
- 姓全大写，名缩写（无标点）
- 3 人及以下：全部列出
- 超过 3 人：前 3 人 + ", et al."

**示例：**
```
SMITH J, JOHNSON A.
SMITH J, JOHNSON A, WILLIAMS B, et al.
```

## 常见错误

### ❌ 错误示例
```
[1] 张三。人工智能研究。计算机学报，2024.
[2] Smith J. Deep Learning. 2020.
```

### ✅ 正确示例
```
[1] 张三。人工智能研究 [J]. 计算机学报，2024, 47(3): 1-10.
[2] SMITH J. Deep Learning[M]. Cambridge: MIT Press, 2020.
```

## 快速检查清单

- [ ] 作者姓名格式正确（英文姓大写，名缩写）
- [ ] 文献类型标识正确
- [ ] 出版信息完整（出版地、出版者、出版年）
- [ ] 期刊论文包含卷期页码
- [ ] 超过 3 位作者使用"等"或"et al."
- [ ] 标点符号使用正确（中文用全角，英文用半角）
- [ ] 电子资源包含访问日期和 URL
